# 💊 Medicine Reminder System

A simple and user-friendly **Medicine Reminder System** developed in **Java**. This application helps users manage their medication schedule by storing medicine details and reminding them to take medicines at the correct time.

---

## 📌 Features

* ➕ Add new medicines
* ✏️ Update medicine details
* ❌ Delete medicines
* 📋 View all scheduled medicines
* ⏰ Set medicine reminder time
* 🔔 Display reminders when medicine time is reached
* 💾 Store medicine records
* 🖥️ Simple and easy-to-use interface

---

## 🛠️ Technologies Used

* Java
* Java Swing (GUI) *(or JavaFX if applicable)*
* Object-Oriented Programming (OOP)
* File Handling / MySQL *(depending on your project)*

1. Launch the application.
2. Add medicine details.
3. Enter dosage information.
4. Set the reminder time.
5. Save the medicine.
6. The application checks reminder times and alerts the user when it's time to take the medicine.
